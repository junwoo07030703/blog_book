# 3D 책장 프로젝트 구현 완료 보고서

## 개요
**카메라 줌인 효과가 있는 프리미엄 3D 책장 웹사이트**를 구현했습니다.

## 주요 기능 및 구현 내용

### 1. 3D 책장 및 인터랙션
- **CSS 3D Transforms**: `rotateY`, `translateZ` 등을 활용하여 입체적인 책과 선반 구현
- **카메라 줌인 효과**: 책 클릭 시 `transform-origin`과 `scale`을 활용하여 **클릭한 책이 화면 중앙으로 부드럽게 확대**되는 시네마틱 효과 구현 (좌표 편차 문제 해결)
- **상세 보기**: 확대된 상태에서 책 클릭 시 상세 정보 및 독서 감상문 모달 표시

### 2. 디자인 (User Selection)
- **책 렌더링**: 책등이 잘 보이도록 `-20deg` 각도로 배치하여 입체감 강조
- **선반 디자인**: 묵직하고 고급스러운 우드 톤의 `-30px` 두께 선반 유지
- **전체 분위기**: 진중하고 따뜻한 기존의 프리미엄 갤러리 분위기 유지

### 3. 기술 스택 및 성능
- **React + Vite + TypeScript**: 빠르고 안정적인 개발 환경
- **Zero Library 3D**: Three.js 등 무거운 라이브러리 없이 **순수 CSS**로 고성능 3D 효과 구현

## 실행 방법
```bash
npm install
npm run dev
```
